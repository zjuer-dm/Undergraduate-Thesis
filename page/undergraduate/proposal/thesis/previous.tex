    \par {\zihao{4}\bfseries 二、指导教师对文献综述、开题报告、外文翻译的具体要求：}
}

\vspace{1em}
{\zihao{5}
\textbf{对文献综述的具体要求：}
\begin{enumerate}[leftmargin=2em]
    \item 了解深度学习在导航中的相关应用，了解图神经网络及大模型结构的发展现状、及轻步态研究方向国内外发展现状。
    \item 阐述已有视觉语言导航算法及其国内外研究现状。
    \item 比较已有的视觉语言导航算法，阐述大模型在视觉语言导航中的应用及其研究意义。
    \item 做好毕业论文相关知识的准备工作。
\end{enumerate}

\textbf{对开题报告的具体要求：}
\begin{enumerate}[leftmargin=2em]
    \item 分析当前视觉语言导航算法的种类与特点。
    \item 给出视觉语言导航方案，选择核心处理算法及思路。
    \item 对论文要完成的工作有清晰的认识，给出具体的研究方案，预见可能遇到的问题和困难。
    \item 提出论文的研究成果或预期达到的目标。
    \item 安排好论文的进度，对于每个阶段的工作有详细的规划。
\end{enumerate}

\textbf{对外文翻译的具体要求：}
\begin{enumerate}[leftmargin=2em]
    \item 翻译一篇或多篇与视觉语言导航或具身智能相关的英文文献，译文准确、通顺，字数不少于3000字。
    \item 通过翻译加深对相关领域前沿技术的理解，积累专业术语和表达方式。
\end{enumerate}
}
